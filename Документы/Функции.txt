3.	Статус гостя. Атрибуты роли – код статус гостя, статус.
4.	Статус номера гостя. Атрибуты роли – код стутуса номера гостя, статус номера.
5.	Пользователи. Атрибуты – код пользователя, логин пароль, почта, роль.
6.	Акции. Атрибуты – код акции, код типа комнаты, код статуса акции, название акции, номер ацкии, сколько осталось ипользований, начало акции, конец акции, сколько всего использований, описание акции.
7.	Стаус акции. Атрибуты – код статуса акции, статус.
8.	Политика отмены. Атрибуты продукции – код политики отмены, политика отмены, описание.
9.	Номера. Атрибуты пользователя – код номера, код статуса номера, код типа номера, код стутуса номера гостя, код ремонтынх работ, код удобств, номер комнаты, этаж номера.
10.	Тип номера. Атрибуты роли – код типа номера, тип номера.

12.	Статус номера. Атрибуты – код статуса номера, статус.

15.	Расценка. Атрибуты – код расценки, код типа комнаты, код политики отмены, код акции, цена.
16.	Пареметры отеля. Атрибуты продукции – код параметра отеля, код политики хранения персональных данных, название отеля, лотип, страна, регион, город, улица, номер дома, количество этажей, количество номеров, контактная почта, контактный номер телефона, ФИО валдельца, почта владельца, номер телефона владельца.
